# resumepage

# Bienvenido al portfolio de Luciano Moireira, esta pagina actua como un resumen de la la experiencia junto a logros de Luciano.

# Contenido del documento

# 1. Introducción
# El presente documento describe la estructura, funcionalidad y detalles técnicos del proyecto resumepage, 
# una página web personal diseñada para destacar logros académicos y facilitar el proceso de admisión a 
# universidades en Estados Unidos.

# 2. Descripción General del Proyecto
# resumepage es una página web responsiva que resume la trayectoria académica y profesional del usuario. 
# Incluye información clave como educación, experiencia, habilidades, idiomas y proyectos, proporcionando 
# una presentación clara y efectiva para oficiales de admisiones.

# 2.1. Tecnologías Utilizadas
# - HTML5: Estructura y organización del contenido.
# - CSS3: Estilizado y diseño responsivo.
# - JavaScript (si aplica): Interactividad y dinámica de la interfaz.
# - GitHub Pages: Hosting del sitio web.

# 3. Estructura del Proyecto
# El proyecto se organiza en los siguientes archivos y carpetas:
# /resumepage
# ├── index.html         # Estructura principal de la página web
# ├── styles.css         # Estilos generales del sitio
# ├── responsive.css     # Estilos para adaptabilidad en dispositivos
# ├── images/            # Carpeta que contiene las imágenes utilizadas
# └── README.md          # Instrucciones básicas del proyecto

# 4. Descripción de los Archivos Principales
# 4.1. index.html
# Archivo principal que contiene la estructura HTML del sitio. Secciones clave:
# - Encabezado: Nombre y título profesional.
# - Sobre Mí: Breve descripción personal.
# - Idiomas: Listado de idiomas hablados y niveles de dominio.
# - Experiencia: Historial laboral relevante.
# - Educación: Instituciones y logros académicos.
# - Proyectos: Trabajo destacado con enlaces y descripciones.
# - Habilidades: Lenguajes y herramientas dominadas.
# - Contacto: Información de contacto y redes sociales.

# 4.2. styles.css
# Define los estilos visuales del sitio web, incluyendo:
# - Paleta de colores y tipografías.
# - Espaciado y alineación de elementos.
# - Efectos de hover y animaciones sutiles.

# 4.3. responsive.css
# Estilos adaptativos para mejorar la visualización en dispositivos móviles y tablets.

# 5. Instrucciones de Uso
# 5.1. Clonación del Proyecto

#5.2. Visualización Local
